私人执业咨询师成长研究 · 类型系统说明

执业状态类型系统
如何从四个维度产生16种类型

这份说明解释了 AI 画像中“执业状态类型”的分类逻辑，帮助你理解自己的类型代码是如何得出的。

01 · 基本逻辑

这个类型系统的设计灵感来自 MBTI，但聚焦于一个更具体的问题：一个私人执业咨询师，在面对执业困境时，呈现出怎样的状态模式？

我们从四个维度来观察这个问题，每个维度有两个方向，四个维度的两两组合，产生了 2×2×2×2 = 16 种可能的类型。

这16种类型不是评判，也不是诊断。它们只是对"你目前的执业状态"的一种描述性命名，帮助你感到"原来我是这样的"——被看见，而不是被分类。

02 · 四个维度
I / E
困境来源
I = 内省型 E = 处境型
你描述的困境，主要指向内在的质疑（"我是谁、我为什么做这件事"），还是外部的处境（个案来源、收入、孤独）？
S / A
应对方式
S = 停留型 A = 行动型
面对困境时，你倾向于在里面停留、思考、感受，还是倾向于通过行动、改变、解决来应对？
T / C
理解变化
T = 转变型 C = 持守型
这段困境有没有带来对某件事理解的转变？还是困境没有改变你的核心理解，你仍在原有框架内？
D / P
整合程度
D = 消化型 P = 悬置型
这段困境目前是否已有某种程度的整合或和解？还是它仍悬而未决，还在进行中？
03 · 如何读类型代码
ISCD
I = 内省型 S = 停留型 C = 持守型 D = 消化型

四个字母依次对应四个维度（困境来源 → 应对方式 → 理解变化 → 整合程度）。ISCD 守望者：在内在质疑中停留，没有改变核心理解，但与那种不确定性达成了某种和解。

04 · 全部16种类型
ISTD沉淀者
在内在质疑中停留，经历了理解的颠覆，并逐渐找到了新的立足点。
ISTP转折中的人
经历了某个改变理解的时刻，但那个转折还没有完全落地，仍在途中。
ISCD守望者
在内在质疑中停留，没有改变核心理解，但与那种不确定性达成了某种和解。
ISCP悬而未决者
在内在质疑中停留，困境还在，理解未变，与它同行但尚未和解。
IATD整合者
通过行动应对内在质疑，经历了理解的转变，并将它整合进了工作方式。
IATP探索者
用行动回应内在的不确定，理解在变，但还没有找到新的落脚点。
IACD实践者
用行动应对内在质疑，核心理解未变，但已找到与困境共处的方式。
IACP奔跑者
用行动应对内在质疑，困境还在，理解未变，在不停地做但不确定去向哪里。
ESTD见证者
在外部处境的困难中停留，经历了对这份工作的重新理解，并找到了新的方向。
ESTP过渡者
在外部处境的困难中停留，理解在改变，但还在这个过渡的中间地带。
ESCD稳定者
在外部处境的困难中停留，核心理解未变，已与这种处境达成了某种和解。
ESCP独行者
在外部处境的困难中停留，困境还在，理解未变，一个人扛着这一切继续走。
EATD开拓者
用行动应对外部处境，经历了理解的转变，并将新的理解转化成了新的做法。
EATP寻路者
用行动应对外部处境，理解在变，但还没有找到清晰的方向。
EACD建设者
用行动应对外部处境，核心理解未变，已在现有框架内建立了自己的方式。
EACP前行者
用行动应对外部处境，困境还在，理解未变，在不确定中继续向前。
05 · 关于这个系统的说明

这16种类型是基于你填写的问卷内容，由 AI 根据你的回答描述推断得出的。它不是一项经过效度检验的心理测量工具，也不构成任何临床诊断依据。

类型描述的目的是提供一面镜子——让你感到"原来我是这样的"，而不是给你贴一个标签。没有"好"的类型，也没有"坏"的类型，每种类型都是真实的执业状态，都有其内在的合理性。

如果你对自己的类型有疑问，或者感觉描述不太准确，这是正常的——AI 的推断基于有限的文字，无法完整捕捉一个人的全部状态。
